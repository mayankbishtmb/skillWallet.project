# Strategic Product Placement Analysis

### Unveiling Sales Impact with Tableau Visualization

## 📌 Project Overview

This project analyzes how product placement affects sales performance using data visualization in Tableau.
The goal is to identify patterns, trends, and insights that help understand how strategic placement influences customer buying behavior.

## 📊 Tools Used

* Tableau
* Excel / CSV Dataset
* GitHub (for project hosting)

## 📂 Project Files

* Project_Documentation.docx → Full project report
* Tableau Dashboard → Visualization of sales data
* Dataset → Used for analysis
